# flexserverclientupdater

The Flex Server Client Updater is intended for use in the case where a customer has changed their IP or hostname for their Flex server & needs to update client workstations to the new address. 

This tool is comprised of two components, the first is the bat file where you will specify the actions the tool should take, the second is the actual software running the find & replace actions on the PC. 

## IMPORTANT ##

Beause this tool updates the client machine's registry key you will need to run this tool as an administartor on the PC
